# Magictext

## Install

### Install custom tkinter

```bash
sudo python -m pip installl -r requirement.txt
```

#### if tkinter is not install

```bash
sudo apt-get install python-tk
```
## Run python code

```bash
python main.py
```

## build and run
```bash
./build.sh
```
### Run binary

```bash
./build/dist/main/main
```
